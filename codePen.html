CodePen Challenges

VeryEasy.html - This assignment required an unordered list that mentioned the colors of the rainbow. It wasn't that difficult to create, hence the file name "VeryEasy".

Easy.html - This assignment required an HTML page with a main headline, two subheads in their own sections and supporting text for each subhead. Free to choose our own topic for the website, I chose to focus the website about the two things that make up the Marvel Universe. That included superheroes and villains.

Medium.html - This assignment required an HTML page that was fully semantic. Focusing on a different topic was also another requirement so the website wouldn't be repeated, so I decided to make an R&B Website. It included various links to different R&B Artists.

Hard.html - This assignment required an HTML form that would collect different information like name, gender, favorite sport, and what toppings you like on your website.

VeryHard1.html - This assignment required me to get some help from some really nice peers. Creating the horizontal navigation and making one of the items be a dropdown navigation really stumped me.

VeryHard2.html - This assignment was the hardest challenge of the week. It was on Javascript, which I haven't learned yet. Good thing there are a lot of people at RedVentures willing to help.
